Este es un RAG sobre el Mundial FIFA 2026 que su Base de conocimiento es un csv . Se usa Ollama con el modelo gpt-oss:20b-cloud para que la generación de respuestas sea más rápida pero usted puede cambiar el modelo en el archivo main.py por el que desee.
Para correr el proyecto prmeramente deberá crearse un entorno virtual : python -m venv env
Despues instale los requerimientos del proyecto : python install -r requeriments.txt
Abra el Visual Studio o el IDE que utilice ; abra la terminal y corra python main.py
